// CONTEXTO:
Crie o código inicial de um site responsivo para a Igreja Família da Cruz | Cuiabá, com base no PRD e estrutura de identidade visual fornecida (ver abaixo). O projeto já possui bibliotecas UX mobile-first indexadas nos meus arquivos (como Tailwind ou similares). Essas bibliotecas devem ser **usadas obrigatoriamente**.

// OBJETIVO:
Gerar a estrutura HTML + componentes reusáveis (com estilização baseada nas bibliotecas indexadas) para as seguintes seções da página inicial:

1. Header com logo, slogan e horários dos cultos
2. Seção hero com chamada para culto online/presencial
3. Cards com missão, visão e valores
4. Carrossel para mensagens/vídeos (embed YouTube)
5. Feed Instagram embed (últimos 3 posts)
6. Rodapé com endereço, redes sociais e contatos

// RESTRIÇÕES E DIRETRIZES:
- Usar **apenas componentes compatíveis com as bibliotecas mobile-first indexadas**
- Garantir **acessibilidade básica** (alt texts, contraste, navegação via teclado)
- Seguir a paleta de cores e tipografia descrita no PRD
- Estruturar em arquivos/componentes reutilizáveis (se aplicável)
- Manter o estilo acolhedor, bíblico e direto, como nas redes sociais da igreja
- Foco em performance: evite código morto ou carregamento excessivo
- Responsividade obrigatória (mobile-first)
- Sem necessidade de CMS/backend por agora

// NOTAS:
O conteúdo completo com textos, links e campos dinâmicos está no PRD institucional e anexo já indexado. Use campos e nomes de variáveis consistentes com a estrutura proposta em “Coleções” (Configurações do Site, Pastores etc).

// SAÍDA ESPERADA:
Código HTML/JSX + classes de estilização com base nas libs mobile-first
Se possível, modularizado por seção
